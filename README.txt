
# Expense Tracker

Приложение для отслеживания личных расходов с фильтрацией и подсчётом сумм.

## Автор

[Кирилл Чепухин]

## Описание

Приложение позволяет:
* Добавлять расходы с указанием суммы, категории и даты.
* Фильтровать записи по категории и дате.
* Подсчитывать общую сумму расходов за выбранный период.
* Сохранять данные в JSON‑файл.

## Как использовать

1. Запустите приложение: `python main.py`.
2. Заполните поля:
   * **Сумма** — числовое значение (положительное).
   * **Категория** — выберите из списка.
   * **Дата** — в формате `ГГГГ-ММ-ДД`.
3. Нажмите **«Добавить расход»**.
4. Для фильтрации:
   * Выберите категорию в фильтре.
   * Введите дату в формате `ГГГГ-ММ-ДД` в поле фильтра по дате.
   * Нажмите **«Применить фильтры»**.
5. Для подсчёта суммы за период:
   * Укажите начальную и конечную дату в формате `ГГГГ-ММ-ДД`.
   * Нажмите **«Посчитать сумму за период»**.

## Примеры использования

**Тест 1: Добавление расхода**
1. Введите сумму: `1500`.
2. Выберите категорию: `Еда`.
3. Введите дату: `2024-05-15`.
4. Нажмите **«Добавить расход»** → появится сообщение «Успех», запись появится в таблице.

**Тест 2: Фильтрация по категории**
1. Выберите в фильтре категорию `Транспорт`.
2. Нажмите **«Применить фильтры»** → в таблице останутся только расходы категории «Транспорт».

**Тест 3: Подсчёт суммы за период**
1. В поле «Период для подсчёта» укажите: `2024-05-01` и `2024-05-31`.
2. Нажмите **«Посчитать сумму за период»** → отобразится общая сумма расходов за май 2024 года.
